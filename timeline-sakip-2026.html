📆 Export ke Google Calendar

Download file .ics di bawah ini, lalu import ke Google Calendar untuk mendapatkan pengingat otomatis seluruh kegiatan SAKIP 2026.

📅 Buka Google Calendar
1

Klik tombol Download File .ics SAKIP 2026 di atas.

2

Buka Google Calendar → klik ikon ⚙️ (Settings) → pilih Import & Export.

3

Klik Select file from your computer → pilih file .ics yang sudah didownload.

4

Pilih kalender tujuan (misal: "SAKIP 2026") → klik Import.

5

Semua event SAKIP 2026 akan langsung muncul di kalender kamu! ✅

📝 Catatan Penting
  • E-Performance wajib diisi setiap tanggal 5 setiap bulan tanpa terkecuali.
  • Evaluasi SAKIP oleh ITJEN berlangsung Maret–Juni 2026.
  • Evaluasi SAKIP oleh KEMENPAN-RB estimasi 8 Agustus 2026.
  • Upload ESR & E-Performance wajib dilakukan untuk setiap kegiatan yang ditandai.
  • Tanggal EST (Estimasi) dapat berubah sesuai kebijakan KEMENPAN-RB.
